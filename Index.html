Project on Console!

Visual Studio Code is a freeware source-code editor made by Microsoft for Windows, Linux and macOS. Features include support for debugging, syntax highlighting, intelligent code completion, snippets, code refactoring, and embedded Git. Users can change the theme, keyboard shortcuts, preferences, and install extensions that add additional functionality. To set up Visual Studio you have to go to their google website, Install the application, Accept terms and conditions and Install the app itself.

To open the console screen, please right click on the mouse and click on inspect which then shows you the code of the screen you are looking at right now, after that, click on console and voila! you are looking at the console screen :)
